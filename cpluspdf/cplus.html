Brief History and Evolution of C++

C++ is a general-purpose programming language that was developed by Bjarne Stroustrup at Bell Labs in Murray Hill, New Jersey, during the early 1980s. The language was designed as an enhancement to the C language, aiming to provide a more flexible and powerful set of features while retaining the efficiency and performance of C.

Early Development:

  • 1979: Bjarne Stroustrup begins working on "C with Classes," an extension of the C language that introduces object-oriented programming (OOP) concepts.
  • 1983: "C with Classes" is renamed C++ to reflect its evolution from C. The name C++ symbolizes the language's progression, with the "++" operator denoting an increment in C's capabilities.

Significant Versions:

  • C++98: The first standardized version of C++, adopted by the International Organization for Standardization (ISO) in 1998. It formalized the language features and libraries.
  • C++03: A minor revision of C++98, addressing some defects and ambiguities in the language.
  • C++11: Introduced significant enhancements, including auto keyword, range-based for loops, lambda expressions, and smart pointers.
  • C++14: Added minor improvements and bug fixes over C++11.
  • C++17: Brought new features like std::optional, std::variant, and parallel algorithms.
  • C++20: Introduced concepts, ranges, calendar and time zone libraries, and coroutines.
  • C++23: Continues to improve the language with features like improved constexpr, expanded standard library components, and better support for modules.

Key Features of C++

Object-Oriented Programming (OOP):

  • Classes and Objects: C++ supports encapsulation, inheritance, and polymorphism, allowing for the creation of complex systems through the use of classes and objects.
  • Inheritance: Enables the creation of new classes that are based on existing classes, promoting code reusability.
  • Polymorphism: Allows for methods to do different things based on the object that invokes them.

Low-Level Manipulation:

  • Pointers and References: C++ allows direct memory manipulation through pointers and references, providing powerful control over system resources and performance.

Standard Template Library (STL):

  • Containers: Includes various container classes like vectors, lists, maps, and queues that help in efficient data management.
  • Algorithms: Provides a set of algorithms that can be applied to containers, such as sorting and searching.

Type Safety and Templates:

  • Templates: C++ supports function and class templates, allowing for generic programming. This means that you can write code that works with any data type.

Exception Handling:

  • Try, Catch, and Throw: C++ includes a robust mechanism for handling errors through exception handling, allowing for more reliable and maintainable code.

Operator Overloading:

  • Custom Operators: C++ allows operators to be overloaded so that they can work with user-defined classes, making it easier to work with complex data types.

Namespaces:

  • Organizing Code: Namespaces help in organizing code and avoiding name conflicts in large projects.

RAII (Resource Acquisition Is Initialization):

  • Automatic Resource Management: C++ uses RAII to manage resource allocation and deallocation automatically, which helps in avoiding resource leaks.

Importance and Applications of C++

Performance and Efficiency:

C++ is known for its performance and efficiency, making it suitable for system-level programming, real-time applications, and performance-critical systems.

System Software:

Operating Systems: C++ is used in the development of operating systems and system-level applications due to its close-to-hardware capabilities and performance.

Game Development:

Graphics and Games: C++ is widely used in game development for its performance and control over system resources, particularly in graphics-intensive applications.

Embedded Systems:

Hardware Interaction: C++ is used in embedded systems where low-level hardware interaction is required.

High-Performance Computing:

Scientific Computing: The language's performance features make it suitable for applications in scientific computing and simulations.

Finance and Trading Systems:

Real-Time Processing: C++ is employed in high-frequency trading systems and financial applications that require real-time processing and low latency.

Software Development:

Applications and Libraries: Many large-scale applications and libraries are written in C++ due to its efficiency, flexibility, and extensive standard library support.

Cross-Platform Development:

Portability: C++ supports cross-platform development, allowing for the creation of applications that can run on different operating systems with minimal modifications.

Getting Started with C++

1. Overview of Compilers

A compiler is a tool that translates code written in a high-level programming language (like C++) into machine code that the computer can execute. Here are some popular C++ compilers:

  • GCC (GNU Compiler Collection): A widely used open-source compiler for C, C++, and other languages. It is available on multiple platforms, including Windows, macOS, and Linux.
  • Clang: A compiler based on LLVM (Low-Level Virtual Machine). It provides fast compilation and excellent diagnostic messages. It is also available on multiple platforms and is known for its modular architecture.
  • MSVC (Microsoft Visual C++): A compiler provided by Microsoft, part of the Visual Studio suite. It is specifically designed for Windows and integrates well with other Microsoft tools and libraries.

2. Installation Instructions

a. Windows

GCC (via MinGW):

  • Download the MinGW installer from the MinGW website or the MinGW-w64 project.
  • Run the installer and follow the instructions. Make sure to select the mingw32-base and mingw-gcc-g++ packages.
  • Add the path to the MinGW bin directory (e.g., C:\MinGW\bin) to your system’s PATH environment variable.

Clang:

  • Download the LLVM installer from the LLVM website.
  • Run the installer and follow the instructions.
  • Ensure the LLVM bin directory is added to your system’s PATH.

MSVC:

  • Download and install Visual Studio from the Microsoft website.
  • During installation, select the "Desktop development with C++" workload.
  • Follow the installation instructions, and MSVC will be set up as part of Visual Studio.

b. macOS

GCC:

  • Install Homebrew if you haven’t already (visit Homebrew’s website for instructions).
  • Open Terminal and run brew install gcc.
  • Homebrew will install GCC and make it available in your PATH.

Clang:

  • Clang comes pre-installed with Xcode Command Line Tools.
  • Install Xcode Command Line Tools by running xcode-select --install in Terminal.

MSVC:

  • MSVC is not natively available on macOS. You might consider using a cross-platform development environment or alternative options like remote development tools.

c. Linux

GCC:

  • Most Linux distributions come with GCC pre-installed. You can check this by running gcc --version in Terminal.
  • If not installed, use the package manager for your distribution. For example, on Ubuntu, run sudo apt-get install build-essential.

Clang:

  • Install Clang using your distribution’s package manager. For example, on Ubuntu, run sudo apt-get install clang.

MSVC:

  • MSVC is not available on Linux. Consider using GCC or Clang for C++ development on Linux.

Basic Syntax and Structure in C++

1. Program Structure

a. #include Directive

The #include directive is used to include header files in your program. These header files contain declarations for functions, classes, and objects that you can use in your code.

#include <iostream> // For standard input/output stream
#include "myheader.h" // For user-defined header files

<iostream> is a standard library header that provides functionality for input and output. "myheader.h" is an example of a user-defined header file that you might include in your program.

b. main() Function

The main() function is the entry point of a C++ program. It is where the execution of the program begins. Every C++ program must have a main() function.

int main() {
// code to be executed
return 0; // Return 0 to indicate successful execution
}

int indicates that the main() function returns an integer value. return 0; signifies successful execution of the program. Other return values can be used to indicate different statuses or errors.

c. Basic Input/Output using cin and cout

C++ provides standard objects for input and output operations:

  • cout for output to the console
  • cin for input from the console
#include <iostream> // Include iostream for input/output

int main() {
std::cout << "Hello, World!" << std::endl; // Output text to console
int number;
std::cin >> number; // Input number from console
std::cout << "You entered: " << number << std::endl; // Output the entered number
return 0;
}

std::cout is used to output data to the console. << is the insertion operator. std::cin is used to receive input from the console. >> is the extraction operator. std::endl is used to insert a newline character and flush the output buffer.

2. Comments

a. Single-Line Comments (//)

Single-line comments begin with // and continue to the end of the line. They are used for brief comments or explanations.

// This is a single-line comment
int x = 10; // Initialize x with 10

b. Multi-Line Comments (/* ... */)

Multi-line comments start with /* and end with */. They can span multiple lines and are useful for longer explanations or temporary code disablement.

/*
* This is a multi-line comment.
* It can span multiple lines.
*/
int y = 20;

Multi-line comments cannot be nested within each other. They are often used for detailed descriptions or to temporarily disable blocks of code during debugging.

Data Types and Variables in C++

1. Primitive Data Types

Primitive data types are the basic data types provided by C++ that represent single values. Here are the fundamental primitive data types:

  • int: Represents integer values.
    Size: Typically 4 bytes (32 bits) but can vary based on the system.
    Example: int age = 25;
  • float: Represents floating-point numbers (numbers with decimal points).
    Size: Typically 4 bytes (32 bits).
    Precision: Up to 6 decimal places.
    Example: float temperature = 36.6f;
  • double: Represents double-precision floating-point numbers.
    Size: Typically 8 bytes (64 bits).
    Precision: Up to 15 decimal places.
    Example: double pi = 3.141592653589793;
  • char: Represents a single character.
    Size: Typically 1 byte (8 bits).
    Example: char grade = 'A';
  • bool: Represents Boolean values, which can be either true or false.
    Size: Typically 1 byte (8 bits).
    Example: bool isPassed = true;

2. Type Modifiers

Type modifiers are keywords used to modify the meaning of basic data types. They allow you to define the size and signedness of a variable.

  • short: Reduces the size of an integer type.
    Size: Typically 2 bytes (16 bits).
    Example: short smallNumber = 32000;
  • long: Increases the size of an integer type.
    Size: Typically 4 bytes (32 bits) or 8 bytes (64 bits), depending on the system.
    Example: long largeNumber = 100000L;
  • signed: Specifies that the variable can hold both positive and negative values. It is the default for integer types.
    Example: signed int balance = -500;
  • unsigned: Specifies that the variable can only hold non-negative values. It effectively doubles the range of positive values compared to its signed counterpart.
    Example: unsigned int age = 30;

3. Variable Declaration and Initialization

Variable Declaration: In C++, variables must be declared before they are used. Declaration tells the compiler the type and name of the variable.

Variable Initialization: Initialization involves assigning an initial value to the variable at the time of declaration.

#include <iostream>

int main() {
    // Primitive Data Types
    int age = 25;               // Integer
    float height = 5.9f;        // Floating-point
    double pi = 3.1415926535;   // Double precision floating-point
    char grade = 'A';           // Character
    bool isPassed = true;       // Boolean

    // Type Modifiers
    short smallNumber = 32000;  // Short integer
    long largeNumber = 100000L; // Long integer
    signed int balance = -500;  // Signed integer
    unsigned int ageUnsigned = 30; // Unsigned integer

    // Output values
    std::cout << "Age: " << age << std::endl;
    std::cout << "Height: " << height << std::endl;
    std::cout << "PI: " << pi << std::endl;
    std::cout << "Grade: " << grade << std::endl;
    std::cout << "Passed: " << isPassed << std::endl;
    std::cout << "Small Number: " << smallNumber << std::endl;
    std::cout << "Large Number: " << largeNumber << std::endl;
    std::cout << "Balance: " << balance << std::endl;
    std::cout << "Unsigned Age: " << ageUnsigned << std::endl;

    return 0;
}

Key Points:

  • The size of data types can vary between different systems and compilers.
  • Floating-point types (float and double) are used for representing numbers with fractional parts.
  • char is used for single characters and typically occupies 1 byte of memory.
  • bool is used for boolean values, taking up 1 byte but storing true or false.
  • Type modifiers adjust the size and sign of variables to suit different needs.

C++ Operators

1. Arithmetic Operators

Arithmetic operators are used to perform basic arithmetic operations.

Addition (+): Adds two operands.

int a = 5, b = 3;
int sum = a + b; // sum = 8

Subtraction (-): Subtracts the second operand from the first.

int difference = a - b; // difference = 2

Multiplication (*): Multiplies two operands.

int product = a * b; // product = 15

Division (/): Divides the first operand by the second.

int quotient = a / b; // quotient = 1

Modulus (%): Returns the remainder of the division of the first operand by the second.

int remainder = a % b; // remainder = 2

2. Relational Operators

Relational operators are used to compare two values and determine their relationship.

Equal to (==): Checks if two operands are equal.

bool isEqual = (a == b); // isEqual = false

Not equal to (!=): Checks if two operands are not equal.

bool isNotEqual = (a != b); // isNotEqual = true

Greater than (>): Checks if the first operand is greater than the second.

bool isGreater = (a > b); // isGreater = true

Less than (<): Checks if the first operand is less than the second.

bool isLess = (a < b); // isLess = false

Greater than or equal to (>=): Checks if the first operand is greater than or equal to the second.

bool isGreaterOrEqual = (a >= b); // isGreaterOrEqual = true

Less than or equal to (<=): Checks if the first operand is less than or equal to the second.

bool isLessOrEqual = (a <= b); // isLessOrEqual = false

3. Logical Operators

Logical operators are used to combine or invert boolean expressions.

Logical AND (&&): Returns true if both operands are true.

bool andResult = (a > 2 && b < 4); // andResult = true

Logical OR (||): Returns true if at least one of the operands is true.

bool orResult = (a > 2 || b > 4); // orResult = true

Logical NOT (!): Inverts the boolean value of the operand.

bool notResult = !(a > 2); // notResult = false

4. Assignment Operators

Assignment operators are used to assign values to variables.

Assignment (=): Assigns the value on the right to the variable on the left.

int x = 10; // x = 10

Addition assignment (+=): Adds the right operand to the left operand and assigns the result to the left operand.

x += 5; // x = x + 5; x = 15

Subtraction assignment (-=): Subtracts the right operand from the left operand and assigns the result to the left operand.

x -= 3; // x = x - 3; x = 12

Multiplication assignment (*=): Multiplies the left operand by the right operand and assigns the result to the left operand.

x *= 2; // x = x * 2; x = 24

Division assignment (/=): Divides the left operand by the right operand and assigns the result to the left operand.

x /= 4; // x = x / 4; x = 6

Modulus assignment (%=): Takes the modulus of the left operand by the right operand and assigns the result to the left operand.

x %= 4; // x = x % 4; x = 2

5. Increment and Decrement Operators

Increment and decrement operators are used to increase or decrease the value of a variable by one.

Increment (++): Increases the value of the operand by one.

Prefix (++x): Increments the value before using it.

int y = 5;
int z = ++y; // y = 6, z = 6

Postfix (x++): Increments the value after using it.

int y = 5;
int z = y++; // y = 6, z = 5

Decrement (--): Decreases the value of the operand by one.

Prefix (--x): Decrements the value before using it.

int y = 5;
int z = --y; // y = 4, z = 4

Postfix (x--): Decrements the value after using it.

int y = 5;
int z = y--; // y = 4, z = 5

6. Bitwise Operators

Bitwise operators perform operations on the binary representations of numbers.

AND (&): Performs a bitwise AND operation.

int a = 12; // 1100 in binary
int b = 7; // 0111 in binary
int result = a & b; // result = 4 (0100 in binary)

OR (|): Performs a bitwise OR operation.

int result = a | b; // result = 15 (1111 in binary)

XOR (^): Performs a bitwise XOR operation.

int result = a ^ b; // result = 11 (1011 in binary)

Complement (~): Performs a bitwise NOT operation.

int result = ~a; // result = -13 (inverts all bits of 12)

Left Shift (<<): Shifts the bits of the first operand to the left by the number of positions specified by the second operand.

int result = a << 2; // result = 48 (1100 << 2 = 110000 in binary)

Right Shift (>>): Shifts the bits of the first operand to the right by the number of positions specified by the second operand.

int result = a >> 2; // result = 3 (1100 >> 2 = 0011 in binary)

1. Conditional Statements

a. if, else if, else

Conditional statements let you execute certain blocks of code based on a condition.

if (condition) {
            // Code to execute if condition is true
        } else if (another_condition) {
            // Code to execute if another_condition is true
        } else {
            // Code to execute if none of the above conditions are true
        }

Example:

#include <iostream>
        using namespace std;
        
        int main() {
            int number = 15;
        
            if (number > 0) {
                cout << "The number is positive." << endl;
            } else if (number < 0) {
                cout << "The number is negative." << endl;
            } else {
                cout << "The number is zero." << endl;
            }
        
            return 0;
        }

b. switch Statement

The switch statement is used to select one of many code blocks to be executed.

switch (expression) {
            case value1:
                // Code to execute if expression equals value1
                break;
            case value2:
                // Code to execute if expression equals value2
                break;
            default:
                // Code to execute if expression doesn't match any case
        }

Example:

#include <iostream>
        using namespace std;
        
        int main() {
            int day = 3;
        
            switch (day) {
                case 1:
                    cout << "Monday" << endl;
                    break;
                case 2:
                    cout << "Tuesday" << endl;
                    break;
                case 3:
                    cout << "Wednesday" << endl;
                    break;
                case 4:
                    cout << "Thursday" << endl;
                    break;
                case 5:
                    cout << "Friday" << endl;
                    break;
                case 6:
                    cout << "Saturday" << endl;
                    break;
                case 7:
                    cout << "Sunday" << endl;
                    break;
                default:
                    cout << "Invalid day" << endl;
            }
        
            return 0;
        }

2. Loops

a. for Loop

The for loop is used when you know in advance how many times you need to execute a statement or a block of statements.

for (initialization; condition; update) {
            // Code to execute repeatedly
        }

Example:

#include <iostream>
        using namespace std;
        
        int main() {
            for (int i = 0; i < 5; ++i) {
                cout << "Iteration " << i << endl;
            }
        
            return 0;
        }

b. while Loop

The while loop is used when you need to repeat a block of code as long as a condition remains true.

while (condition) {
            // Code to execute repeatedly
        }

Example:

#include <iostream>
        using namespace std;
        
        int main() {
            int count = 0;
        
            while (count < 5) {
                cout << "Count is " << count << endl;
                ++count;
            }
        
            return 0;
        }

c. do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the code block will be executed at least once.

do {
            // Code to execute repeatedly
        } while (condition);

Example:

#include <iostream>
        using namespace std;
        
        int main() {
            int count = 0;
        
            do {
                cout << "Count is " << count << endl;
                ++count;
            } while (count < 5);
        
            return 0;
        }

3. Break and Continue Statements

a. break Statement

The break statement is used to exit from a loop or switch statement prematurely.

#include <iostream>
        using namespace std;
        
        int main() {
            for (int i = 0; i < 10; ++i) {
                if (i == 5) {
                    break; // Exit the loop when i equals 5
                }
                cout << "i = " << i << endl;
            }
        
            return 0;
        }

b. continue Statement

The continue statement is used to skip the current iteration of a loop and proceed to the next iteration.

#include <iostream>
        using namespace std;
        
        int main() {
            for (int i = 0; i < 10; ++i) {
                if (i % 2 == 0) {
                    continue; // Skip even numbers
                }
                cout << "i = " << i << endl;
            }
        
            return 0;
        }

C++ Functions

1. Function Declaration and Definition

Function Declaration: A function declaration (also known as a function prototype) tells the compiler about the function's name, return type, and parameters, but does not define what the function does.

return_type function_name(parameter_list);

Example:

int add(int, int);  // Function declaration

Function Definition: A function definition provides the actual body of the function, specifying what the function does.

return_type function_name(parameter_list) {
            // function body
        }

Example:

int add(int a, int b) {
            return a + b;
        }

Usage:

int main() {
            int result = add(5, 3);  // Calling the function
            std::cout << "Sum: " << result << std::endl;
            return 0;
        }

2. Function Overloading

Function overloading allows you to have multiple functions with the same name but different parameter lists. The correct function is chosen based on the number and type of arguments passed.

return_type function_name(parameters);

Example:

#include <iostream>
        
        int add(int a, int b) {
            return a + b;
        }
        
        double add(double a, double b) {
            return a + b;
        }
        
        int main() {
            std::cout << "Sum of integers: " << add(5, 3) << std::endl;
            std::cout << "Sum of doubles: " << add(5.5, 3.3) << std::endl;
            return 0;
        }

3. Default Arguments

Default arguments allow you to omit certain arguments when calling a function. If an argument is omitted, the default value specified in the function declaration or definition is used.

return_type function_name(parameter_list = default_value);

Example:

#include <iostream>
        
        int add(int a, int b = 10) {
            return a + b;
        }
        
        int main() {
            std::cout << "Sum with two arguments: " << add(5, 3) << std::endl;
            std::cout << "Sum with one argument: " << add(5) << std::endl;  // Uses default value for b
            return 0;
        }

4. Recursion

Recursion occurs when a function calls itself to solve a problem. A recursive function must have a base case to end the recursion.

return_type function_name(parameters) {
            if (base_condition) {
                return base_case_value;
            } else {
                return function_name(modified_parameters);
            }
        }

Example:

#include <iostream>
        
        int factorial(int n) {
            if (n <= 1) {
                return 1;  // Base case
            } else {
                return n * factorial(n - 1);  // Recursive case
            }
        }
        
        int main() {
            std::cout << "Factorial of 5: " << factorial(5) << std::endl;
            return 0;
        }

5. Inline Functions

Inline functions suggest to the compiler to insert the function's code directly into the calling code rather than making a function call. This can reduce function call overhead, but may increase the size of the binary if overused.

inline return_type function_name(parameters) {
            // function body
        }

Example:

#include <iostream>
        
        inline int square(int x) {
            return x * x;
        }
        
        int main() {
            std::cout << "Square of 5: " << square(5) << std::endl;
            return 0;
        }

Note: The inline keyword is a suggestion to the compiler. The compiler may choose to ignore it and use a regular function call if it deems appropriate.

Arrays

1. Declaration, Initialization, and Access

Declaration: Arrays are declared by specifying the type of elements and the number of elements.

int arr[5]; // Declares an array of 5 integers

Initialization: Arrays can be initialized at the time of declaration or afterward.

int arr[5] = {1, 2, 3, 4, 5}; // Initializes with values

If not initialized, the array elements will contain garbage values.

Access: Elements are accessed using indices, starting from 0.

int firstElement = arr[0]; // Accesses the first element
            arr[2] = 10; // Sets the third element to 10

Example:

#include <iostream>
            using namespace std;
            
            int main() {
                int arr[5] = {1, 2, 3, 4, 5};
            
                // Access and print elements
                for (int i = 0; i < 5; i++) {
                    cout << "Element at index " << i << " is " << arr[i] << endl;
                }
                
                return 0;
            }

2. Multi-dimensional Arrays

Declaration: Multi-dimensional arrays can be declared similarly to single-dimensional arrays but with multiple sizes.

int matrix[3][4]; // Declares a 2D array (3 rows x 4 columns)

Initialization: Multi-dimensional arrays can be initialized using nested braces.

int matrix[2][3] = {
                {1, 2, 3},
                {4, 5, 6}
            };

Access: Elements are accessed using multiple indices.

int value = matrix[1][2]; // Accesses element in second row, third column
            matrix[0][0] = 10; // Sets element in the first row, first column to 10

Example:

#include <iostream>
            using namespace std;
            
            int main() {
                int matrix[2][3] = {
                    {1, 2, 3},
                    {4, 5, 6}
                };
            
                // Access and print elements
                for (int i = 0; i < 2; i++) {
                    for (int j = 0; j < 3; j++) {
                        cout << "Element at [" << i << "][" << j << "] is " << matrix[i][j] << endl;
                    }
                }
                
                return 0;
            }

C++ Pointers and References

1. Pointers

1.1 Declaration, Initialization, and Usage

Declaration: A pointer is a variable that holds the memory address of another variable. To declare a pointer, use the * symbol.

int* ptr; // Declaration of a pointer to an integer

Initialization: Pointers are often initialized to the address of a variable using the address-of operator (&).

int var = 10;   // Regular integer variable
            int* ptr = &var; // Pointer initialized to the address of var

Usage: To access the value pointed to by the pointer, use the dereference operator (*).

cout << *ptr; // Outputs the value of var, which is 10

To change the value of var through the pointer:

*ptr = 20; // Changes the value of var to 20

1.2 Pointer Arithmetic

Increment and Decrement: Pointers can be incremented or decremented to point to the next or previous memory location of the data type they are pointing to.

int arr[] = {1, 2, 3, 4};
            int* ptr = arr;
            
            cout << *ptr;   // Outputs 1
            ptr++;          // Move to the next integer in the array
            cout << *ptr;   // Outputs 2

Pointer Arithmetic Example:

int* ptr = arr;
            for (int i = 0; i < 4; ++i) {
                cout << *(ptr + i) << " "; // Outputs 1 2 3 4
            }

1.3 Dynamic Memory Allocation

Using new and delete: You can allocate memory dynamically on the heap using new and release it using delete.

// Allocating memory for a single integer
            int* ptr = new int;
            *ptr = 30;
            
            cout << *ptr; // Outputs 30
            
            delete ptr; // Freeing the allocated memory
            
            // Allocating memory for an array of integers
            int* arr = new int[5];
            for (int i = 0; i < 5; ++i) {
                arr[i] = i * 10;
            }
            
            for (int i = 0; i < 5; ++i) {
                cout << arr[i] << " "; // Outputs 0 10 20 30 40
            }
            
            delete[] arr; // Freeing the allocated memory for the array

2. References

2.1 Declaration and Usage

Declaration: A reference is an alias for another variable. It is declared using the & symbol.

int var = 10;
            int& ref = var; // ref is a reference to var

Usage: You can use the reference like a regular variable.

ref = 20; // Changes the value of var to 20
            cout << var; // Outputs 20

2.2 Reference vs. Pointer

Syntax and Usage:

Pointers can be reassigned to point to different variables and can be null (i.e., not pointing to any variable).

int a = 10;
            int b = 20;
            int* ptr = &a; // Pointer to a
            ptr = &b;      // Pointer can now point to b

References must be initialized when declared and cannot be null. Once a reference is set to a variable, it cannot be changed to refer to another variable.

int& ref = a;  // Reference to a
            // ref = b;    // Error: Cannot reassign a reference to another variable

Null Safety:

Pointers can be null and require checks to avoid dereferencing null pointers.

int* ptr = nullptr; // Null pointer
            if (ptr) {
                cout << *ptr; // Safe to dereference
            }

References cannot be null and must always refer to a valid object.

int& ref = a; // Must be initialized to a valid variable

2.3 Example of Reference in Function Arguments

void increment(int& num) {
                num++;
            }
            
            int main() {
                int value = 5;
                increment(value); // Passes value by reference
                cout << value; // Outputs 6
                return 0;
            }

6. Abstraction and Encapsulation

Abstraction

Abstraction: Hiding the implementation details and showing only the essential features of the object.

class AbstractClass {

        };
        
        class ConcreteClass : public AbstractClass {
        public:

            }
        };

Encapsulation

Encapsulation: Bundling data and methods that operate on the data into a single unit (class), and restricting access to some of the object's components.

class EncapsulatedClass {
        private:
            int privateData;
        
        public:
            void setPrivateData(int value) {
                privateData = value;
            }
        
            int getPrivateData() {
                return privateData;
            }
        };
        
        int main() {
            EncapsulatedClass obj;
            obj.setPrivateData(100);
            cout << obj.getPrivateData() << endl;
        
            return 0;
        }
